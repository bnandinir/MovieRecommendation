Movie Recommendation System

Project Description

This is a Movie Recommendation System developed using Python and Django. The application recommends movies to users based on their interests and preferences.

Features

- User-friendly interface
- Movie recommendations
- Search functionality
- Django-based web application

Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite

How to Run

1. Clone the repository.
2. Install the required packages.
3. Run:
   python manage.py runserver
4. Open the browser and visit:
   http://127.0.0.1:8000/
